Features

  • Command-line integration with script and binary execution
  • Visual workflow builder for automation and orchestration
  • Live and manual modes for flexible testing
  • Request/response manipulation and spoofing tools

  • Audit support and policy enforcement features

Usage

  • Launch DevSecBox and design your workflow using the visual editor.
  • Select between 'Live' or 'Manual' mode depending on your testing context.
  • Define triggers and associate them with commands, scripts, or programs using the built-in command adapter.
  • Use request/response tools to capture, modify, or spoof traffic for in-depth testing scenarios.
  • Integrate JSON processing tasks to handle API responses or configuration files.
  • Review results, generate insights, and share them with relevant teams to support secure development practices.
